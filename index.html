Squirm

Scala API for JQuery Mobile

Download this project as a .zip file Download this project as a tar.gz file

Overview

Squirm is simple Scala API for dynamic generation of mobile websites using JQuery Mobile framework. Since seeing is believing let's take a look at few examples:

Page with the listview

Page("Squirm Test", "ListView")(
     ListView( 
        LinkListItem( "Collapsible Set", url("/collapsible") ),
        LinkListItem( "Form", url("/form") )
     )   
)

Page with collapsible set

    Page("Squirm Test", "Collapsble")(
        CollapsibleSet(
               Collapsible( "Collapsible", 
                    ListView( 
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" )
                    )
               ),   
               Collapsible( "Collapsible", 
                    ListView( 
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" )
                    )
               ),    
               Collapsible( "Collapsible", 
                    ListView( 
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" ),
                       PropertyListItem( "name", "value" )
                    )
               )    
         )
    )

Page with the form

    Page("Squirm Test", "Form")(

        Form( url("/login") )(     

            Field( title = "User Name:", id = "user" ),
            PasswordField( title = "Password:", id = "pswd" ),
            SubmitButton("Login")

        )

    )

Design

In Squirm each JQuery Mobile element is mapped to a Component, which can then be rendered using it's 'render' method.

Scalatra Support

Squirm comes with Scalatra support. Adding SquirmSupport trait to Scalatra servlet allows simply returning Squirm components for automatic rendering.
